# Assets

This folder should contain your app icons and splash screens.

When you run `npm start` for the first time, Expo will generate default placeholder images.

## Required Files

- `icon.png` - App icon (1024x1024 px)
- `splash.png` - Splash screen (1284x2778 px recommended)
- `adaptive-icon.png` - Android adaptive icon (1024x1024 px)
- `favicon.png` - Web favicon (48x48 px or larger)

## Generate Assets

You can use the Payy logo from the root directory (`../payy.png`) or create custom icons.

For professional icons, consider using:
- https://www.appicon.co/
- https://makeappicon.com/
